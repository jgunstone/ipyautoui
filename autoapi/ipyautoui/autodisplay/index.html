ipyautoui.autodisplay#

displayfile is used to display certain types of files. The module lets us preview a file, open a file, and open its directory.

Example

::#

from ipyautoui.constants import load_test_constants from ipyautoui.displayfile import DisplayFile, Markdown import ipywidgets as w

DIR_FILETYPES = load_test_constants().DIR_FILETYPES

fpths = list(pathlib.Path(DIR_FILETYPES).glob(“*”))

# single file d = DisplayFile(fpths[7]) display(d)
